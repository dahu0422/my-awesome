# CSS 权重

CSS 权重决定当多个选择器作用于同一个元素时，哪个样式优先生效。权重越高，优先级越高。

CSS 权重通常用四个等级来表示，分别是：

| 选择器类型           | 权重值（从高到低） |
| -------------------- | ------------------ |
| 行内样式             | 1,0,0,0            |
| ID 选择器            | 0,1,0,0            |
| 类、伪类、属性选择器 | 0,0,1,0            |
| 元素、伪元素选择器   | 0,0,0,1            |

计算规则：

- 权重从左到右比较，前面位数大的优先级高；
- 继承的样式、通配符选择器权重为 0；
- !important 并不改变权重，但会让样式强制生效；
